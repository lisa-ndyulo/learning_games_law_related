# Build a Government (South Africa) 🇿🇦

A phone-friendly classroom game to teach learners:
1. What a constitution is.
2. What it does.
3. How branches of government work and hold one another accountable.

## Run locally

```bash
python3 -m http.server 4173
```

Then open:

- `http://localhost:4173/`

## Run on GitHub Pages

1. Push this repository to GitHub.
2. In **Settings → Pages**, set source to your main branch and root (`/`).
3. Save, wait for deployment, and open the published URL.

Because this project uses a root `index.html`, it is GitHub Pages friendly with no build step.
